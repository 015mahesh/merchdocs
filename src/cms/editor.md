---
title: Using the Editor
---


The editor gives you the ability to enter and format while working in a “What You See Is What You Get” view of the content. If you prefer to work directly with the underlying HTML code, you can easily change modes. The editor can be used to create content for [pages]({{ site.baseurl }}{% link cms/pages.md %}), [blocks]({{ site.baseurl }}{% link cms/blocks.md %}), and [product descriptions]({{ site.baseurl }}{% link catalog/product-content.md %}). When working on product detail, the editor is accessed by clicking <span class="btn">Show / Hide Editor</span>.

{: .bs-callout .bs-callout-info}
TinyMCE 4 is now the default WYSIWYG editor. TinyMCE 3 is now deprecated, but can be enabled in the [configuration]({{ site.baseurl }}{% link cms/editor-configure.md %}) if required for previous customizations.
